**Design Philosophy:** Lean, focused, and conversational.
**Personality:** Efficient & Direct.
**Best for:** Users who value speed and an uncluttered interface, similar to modern chat applications.

**Key Characteristics:** - **Layout:** Centered chat window, fixed bottom input. - **Density:** Spacious with clear message separation. - **Navigation:** Implicit (conversational flow). - **Primary Action:** Direct text input and send. - **Visual Weight:** Very minimal, relying on whitespace.
